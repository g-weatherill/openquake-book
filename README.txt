# Content

The Modellers' Toolkit Book (version 0.1 - alpha)

# Purpose
The OpenQuake Book, the OpenQuake User Manual and the Modellers' Toolkit Book
the are intended to provide version controlled, open source documentation for 
Hazard and Risk input model prepararation and calculation related to the 
OpenQuake project.
